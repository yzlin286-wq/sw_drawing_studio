# GB A4 横式 .drwdot 模板规格

本文件是 `templates/gb_a4_landscape.drwdot` 的设计源单。所有坐标使用 SolidWorks API 的国际单位制（米 m）。
图纸方向：A4 横式（landscape），宽 297 mm × 高 210 mm。

---

## A. A4 横式图框（米单位）

| 项 | 坐标（左下角原点 = (0, 0)） |
|---|---|
| 纸张 | 0 ~ 0.297 × 0 ~ 0.210 m（A4 横式） |
| 外框 | (0.010, 0.010) — (0.287, 0.200) |
| 内框 | (0.025, 0.015) — (0.282, 0.195)（左 25 mm 装订 + 上下右各 5 mm） |
| 标题栏外框 | (0.102, 0.015) — (0.282, 0.055)（180 × 40 mm，右下贴齐内框） |
| 标题栏横线 | y = 0.028, y = 0.041 |
| 标题栏竖线 | x = 0.137, 0.182, 0.227, 0.252 |

19 条线列表（外框 4 + 内框 4 + 标题栏外框 4 + 横线 2 + 竖线 4 + 复用 1 = 19；其中标题栏底/右边与内框共线，逻辑上独立绘制以便单独图层化）：

```

标题栏外框: (0.102,0.015)→(0.282,0.015)→(0.282,0.055)→(0.102,0.055)→(0.102,0.015)
横线:  (0.102,0.028)→(0.282,0.028) ; (0.102,0.041)→(0.282,0.041)
竖线:  (0.137,0.015)→(0.137,0.055) ; (0.182,0.015)→(0.182,0.055)
       (0.227,0.015)→(0.227,0.055) ; (0.252,0.015)→(0.252,0.055)
```

## B. 标题栏 5×3 = 15 格表

行索引：row 0 = 顶行（y=0.043），row 1 = 中行（y=0.030），row 2 = 底行（y=0.017）。
列索引：col 0 ~ 4，对应 x = 0.104 / 0.139 / 0.184 / 0.229 / 0.254（cell 中心点）。

| cell_id | x | y | label / 占位文本 | prp_key |
|---|---|---|---|---|
| [0,0] | 0.104 | 0.043 | 机型     | 机型 |
| [0,1] | 0.139 | 0.043 | 品名     | 品名 |
| [0,2] | 0.184 | 0.043 | 图号     | 图号 |
| [0,3] | 0.229 | 0.043 | 类别     | 类别 |
| [0,4] | 0.254 | 0.043 | 数量     | 数量 |
| [1,0] | 0.104 | 0.030 | 材质     | 材质 |
| [1,1] | 0.139 | 0.030 | 表面处理 | 表面处理 |
| [1,2] | 0.184 | 0.030 | 比例     | 比例 |
| [1,3] | 0.229 | 0.030 | 重量     | 重量 |
| [1,4] | 0.254 | 0.030 | 单位     | UNIT_OF_MEASURE |
| [2,0] | 0.104 | 0.017 | 设计     | 设计 |
| [2,1] | 0.139 | 0.017 | 日期     | 日期 |
| [2,2] | 0.184 | 0.017 | $PRP:"图号" | 图号 |
| [2,3] | 0.229 | 0.017 | $PRP:"类别" | 类别 |
| [2,4] | 0.254 | 0.017 | $PRP:"数量" | 数量 |

说明：
- 前 12 格写中文标签 + `$PRP:"<key>"` 占位符让 SW 自动替换为 CustomProperty 值。
- 末 3 格直接放 SolidWorks 的 PRP 链接占位 `$PRP:"<key>"`，渲染时会被替换为对应值。
- 每个 cell 的 (x, y) 是文本插入定位点（`Annotation.SetPosition2(x, y, 0)`）。

## C. 字高映射

| 用途 | 高度 (m) | 高度 (mm) |
|---|---|---|
| 默认 Annotation Default | 0.005   | 5.0 |
| 标题栏小字              | 0.0035  | 3.5 |
| 视图标签                | 0.005   | 5.0 |

API：`SetUserPreferenceDoubleValue(89, h)`，其中 ID `89` = `swDetailingDimensionTextHeight`。
应用顺序：先在 `SldWorks.Application` 级别设一次，`NewDocument` 后再在 `IModelDoc2` 级别设一次。

## D. 5 图层定义

| 名称 | 描述 | 颜色 (RGB int) | 颜色 (BGR int 实际传值) | 线型 (style_e) | 线宽 (m) | weight_e（SW 1~9 对照） |
|---|---|---|---|---|---|---|
| 粗实 | thick_solid | 0x000000 | 0x000000 | 0 (Solid)      | 0.0007  | 6 ≈ 0.7 mm |
| 细实 | thin_solid  | 0x000000 | 0x000000 | 0 (Solid)      | 0.00035 | 2 ≈ 0.35 mm |
| 虚线 | dashed      | 0x000000 | 0x000000 | 1 (Dashed)     | 0.00035 | 2 |
| 点划 | center      | 0xFF0000 | 0x0000FF | 4 (ChainThick) | 0.00035 | 2 |
| 中心 | centerline  | 0xFF0000 | 0x0000FF | 4              | 0.00025 | 1 ≈ 0.25 mm |

API：`LayerMgr.AddLayer(name, desc, color_int_BGR, style_e, weight_e)`

⚠️ 颜色值是 **BGR int**，不是 RGB：红色 RGB=0xFF0000 → BGR=0x0000FF。
SolidWorks 的 line weight 是枚举值（1~9），不是直接传米；规格表里同时给出物理线宽 (m) 与对应枚举编号便于交叉校核。

## E. 13 个 CustomProperty key

```
机型 / 品名 / 图号 / 类别 / 数量 / 材质 / 表面处理 / 比例 / 重量 / UNIT_OF_MEASURE / 设计 / 日期 / 重量
```

注意：`重量` 出现两次（一次在标题栏中行 [1,3]，对应实重；规格里 13 字段保留是为了与 v5 SubTask 4.x 的字段名映射兼容）。

⚠️ 写值流程：用 `BlockInst.GetUpdateAllParameters()` 拿到 block 实例的参数集合后写值，**不直接 add CustomProperty**。否则模板会被锁死成一次性属性。
